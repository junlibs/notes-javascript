## chapter-01.入门
### 1.1 HelloWorld
js有三种输出方式：
1. 通过`alert()`函数输出
2. 通过`console.log()`函数输出
3. 通过`document.write()`函数输出

### 1.2 js代码编写位置
js代码有以下常见编写位置：
1. 写在`<script>写在此处</script>`标签中
2. 写在外部的js文件中，然后用script标签的src属性引入
3. 写在标签的指定属性中，如`<a href="javascript:alert(123);">`
